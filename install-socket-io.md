# Install Socket.IO Client for Real-time Live Stream Features

Run this command in the fretiko-mobile directory:

```bash
npm install socket.io-client@^4.7.2
```

This adds real-time WebSocket communication for:
- Live comments and reactions
- Viewer count updates  
- Gift sending and receiving
- Live commerce events
- Real-time notifications

The socket service connects to the NestJS WebSocket gateway we built earlier.
